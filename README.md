# Tic-Tac-Toe Game

A simple Tic-Tac-Toe game built using HTML, CSS, and JavaScript.

## Features

- Two-player gameplay
- Winner detection
- Draw detection
- Restart game option
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```
TIC-TAC-TOE GAME/
│
├── index.html
├── style.css
└── app.js
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Start playing the game.
